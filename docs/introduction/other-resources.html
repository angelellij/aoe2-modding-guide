Resources

Discords

  1. Siege Engieneers Discord: This is a community of all around aoe2 modders.
  2. AI Scripters Discord: This is a community of aoe2 modders that specialize on AI scripting.
  3. Random Map (AOE2) Discord: This is a community of aoe2 modders that specialize on random map scripting.
  4. SLX Studio Discord: SLX is a program for creating the building and units sprites. The program was made by Tevious, and the community here is built around that software.

Some guides

  1. AnneHK UI mod reference: Reddit post with some really good images that identify which files affect the UI.
  2. A layout tutorial (JSON): This is a comprehensive guide on UI stuff made by the Chinese community written by cly806 and secondly newtonerdai and uploaded to steam by Minglan_Sheng1016.
