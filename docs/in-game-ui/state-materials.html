State Materials

State materials are an attribute that goes inside widgets. What they do is they indicate a style that the widget should have, and this style depends on the state. The default state is "StateNormal". I'll paste a couple of examples with a short explanation on the right. 

"StateMaterials": {
    "StateNormal": {
        "Material": "ButtonCloseNormal",
        "Font": {
            "FontIndex": 0,
            "PointSize": 38,
            "Style": "Normal",
            "TextColor": {
                "r": 242,
                "g": 210,
                "b": 182,
                "a": 255
            }
            "TextOutlineWidth": 1,
            "TextOutlineColor": {
                "r": 0.1,
                "g": 0.1,
                "b": 0.1,
                "a": 1
            }
        }
    },
    "StateHover": {
        "Material": "FlatColor",
        "Color": {
            "r": 0.25,
            "g": 0.25,
            "b": 0.25,
            "a": 0
        },
        "Font": {
            "FontIndex": 0,
            "PointSize": 38,
            "Style": "Normal",
            "TextColor": {
                "r": 242,
                "g": 210,
                "b": 182,
                "a": 255
            }
        }
    }
}
  1. There are a lot of states possible. The one you will probably use is StateNormal.
  2. Material: It can be a texture or one of the following: FlatColor, TransparentColor,
  3. Color: If material is FlatColor or TransparentColor you can put rgba values to do a solid Background color.
  4. Font: FontIndex: Can be 0, 2 or 3.
  5. Font: PointSize: Size of the Text.
    6. Font: Style: I have not found other options other than "Normal".
  6. Font: TextColor: You need to provide the rgba values like in previous examples.
  7. Font: TextOutlineWidth: If you want an outline.
  8. Font: TextOutlineColor: Change the text outline color using rgba.

State Material as debug

Sometimes you lose track of where widgets are positioned or how big they are. In that case, setting a TransparentColor with some values can help you understand your UI.

Use this code in your widget if you ever get lost (it should paint the widget red). 

"StateMaterials": {
    "StateNormal": {
        "Material": "TransparentColor",
        "Color": {
            "r": 1,
            "g": 0,
            "b": 0,
            "a": 0.5
        }
    }
}
